# TRACE

**Trigger, Record & Analyze Configurable Equipment** — an open-source, dual-MCU bench instrument combining a logic analyzer, SWD/JTAG programmer, and function generator, basic oscilloscope, with a web UI and AI-assisted circuit diagnosis.

> **Status:** In development — schematic complete, PCB layout in progress.

Built around an **RP2350** (timing-critical tasks via PIO) and an **ESP32-S3** (WiFi, web UI, programming), powered over USB-C.
